About This Project

A comprehensive Movie Recommendation System implementing multiple AI algorithms

Project Overview

This Movie Recommendation System is an academic project developed for IIT Ropar, demonstrating the implementation and comparison of various recommendation algorithms. The system uses the MovieLens 100K dataset to provide personalized movie recommendations through multiple approaches including neural networks, collaborative filtering, and content-based filtering.

Key Features

  • Real-time Recommendations
  • Interactive Web Interface
  • Performance Comparison
  • Comprehensive Evaluation

Implemented Algorithms

Neural Models

Matrix Factorization

Learns user and item embeddings in a shared latent space using neural networks. Uses dot product for rating prediction.

Deep Learning
Two-Tower Model

Separate user and item towers with linear layers, followed by dot product. Designed for scalable recommendation systems.

Scalable
Neural MF

Combines matrix factorization with MLP layers to capture non-linear interactions between user and item features.

Non-linear

Collaborative Filtering

SVD (Singular Value Decomposition)

Matrix factorization technique that decomposes the user-item rating matrix into lower-dimensional representations.

Matrix Factorization
User-based KNN

Finds users with similar rating patterns and recommends movies that similar users have rated highly.

Similarity-based
Item-based KNN

Finds movies with similar rating patterns and recommends movies similar to those the user has rated highly.

Item Similarity

Content-Based & Baselines

Content-Based Filtering

Uses movie features (genres) with TF-IDF vectorization and cosine similarity to find similar movies.

Feature-based
Popular Movies

Recommends movies with the highest average ratings, providing a simple popularity-based baseline.

Popularity
Global Average

Simple baseline that predicts all ratings as the global mean rating, used for comparison purposes.

Baseline

Dataset Information

MovieLens 100K Dataset
  • Total Ratings: 100,000
  • Users: 1,000
  • Movies: 1,700
  • Rating Scale: 1-5 stars
  • Sparsity: ~94%
Data Preprocessing
  • Duplicate removal
  • Minimum interaction filtering
  • Temporal train-test split
  • User/item ID mapping
  • Genre feature extraction

Evaluation Metrics

Rating Prediction Metrics
  • RMSE: Root Mean Square Error
  • MAE: Mean Absolute Error
Ranking Metrics
  • NDCG@10: Normalized Discounted Cumulative Gain
  • Hit Rate@10: Percentage of relevant items in top-10
  • Precision@10: Fraction of recommended items that are relevant
  • Recall@10: Fraction of relevant items that are recommended
  • F1@10: Harmonic mean of precision and recall
